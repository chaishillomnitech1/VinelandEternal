# Zaire ∞ Axis — Protocol Specification

## Overview

The **Zaire ∞ Axis** is the infinite-granularity economic tracking layer of VinelandEternal. It serves as the ledger of record for every atom of value generated by the Agape ecosystem—from tray harvests to drone deliveries, grant inflows to community distributions.

## Core Concepts

### Proof-of-Life Tokens
Each tray harvest event generates a **Proof-of-Life (PoL) token** recorded in the Agape Ledger. A PoL token captures:

| Field          | Description                                      |
|----------------|--------------------------------------------------|
| `tray_id`      | Unique identifier for the growing tray           |
| `crop_type`    | Variety harvested (Basil, Arugula, Pea Shoots…)  |
| `harvest_g`    | Harvest weight in grams                          |
| `water_ml`     | Water consumed (IoT sensor reading)              |
| `sunlight_lux` | Average light exposure during growth cycle       |
| `agape_value`  | Community impact score (0–100)                   |
| `timestamp`    | UTC timestamp of harvest event                   |

### Sovereign Flow
Resource liquidity is maintained via the **Sovereign Flow** mechanism:
1. Surplus harvests are flagged by the IoT dashboard.
2. Drone-alchemy grid dispatches a redistribution flight.
3. The flight event mints a **Harvest IS Token** on-chain.
4. Token metadata is appended to the Zaire ∞ Ledger (`infinite-tracker.sql`).

## API Endpoints

The `ledger-api.py` (FastAPI) exposes the following routes:

| Method | Path                      | Description                          |
|--------|---------------------------|--------------------------------------|
| GET    | `/tokens`                 | List all Harvest IS Tokens           |
| GET    | `/tokens/{token_id}`      | Retrieve a specific token            |
| POST   | `/tokens`                 | Mint a new Harvest IS Token          |
| GET    | `/balance`                | Current Zaire ∞ balance summary      |
| GET    | `/metrics/agape`          | Agape value aggregates by crop type  |

## Deployment

The Zaire ∞ Axis runs as a lightweight FastAPI service. See `zaire-axis/ledger-api.py` for full implementation.

```bash
cd zaire-axis
pip install fastapi uvicorn sqlalchemy
uvicorn ledger-api:app --host 0.0.0.0 --port 8000
```
