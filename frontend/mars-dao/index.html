Governance Security

Attack VectorDefense Mechanism

Whale Dominance
100× tokens → control		 Quadratic voting: 100× tokens = 10× votes. Large holders heard, small holders protected.
Flash Loan Attack
Borrow → vote → repay		 Balance at vote time counts. Flash loans repay before vote registers. Zero effect.
Council Capture
Compromise the 5		 3-of-5 multisig + 7-day timelock + community recall. Must compromise 3 independently.
Governance Fatigue
Nobody votes		 Participation rewards make apathy economically irrational. Vote → earn $MARS.
